Course Overview

This course equips students with machine learning (AI) and wireless communication algorithm design capabilities, covering 5G, B5G, and 6G communication systems. Students gain hands-on experience implementing communication algorithms in 5G and O-RAN small base stations on FPGA/RFSoC platforms, mastering cross-domain skills spanning theory, algorithms, and hardware implementation.
